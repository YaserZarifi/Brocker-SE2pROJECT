## ۱. معرفی

### ۱.۱ هدف پروژه

BourseChain یک **سامانه کارگزاری بورس آنلاین** است که امکان خرید و فروش سهام را از طریق پلتفرمی شفاف و امن فراهم می‌کند. تراکنش‌ها روی **بلاکچین خصوصی EVM** ثبت می‌شوند تا قابلیت ردیابی و عدم تکذیب (non-repudiation) تضمین شود.

### ۱.۲ ذینفعان

| ذینفع | نقش | انتظارات |
|-------|-----|-----------|
| **کاربر نهایی (سرمایه‌گذار)** | خرید و فروش سهام | رابط کاربری ساده، اعلان لحظه‌ای، امنیت |
| **ادمین** | مدیریت سهام و کاربران | پنل مدیریت، گزارش‌گیری |
| **تیم توسعه** | نگهداری و توسعه | کد تمیز، مستندات، تست‌ها |
| **دانشگاه** | ارزیابی پروژه | تحویل به موقع، رعایت نیازمندی‌ها |

## ۲. چشم‌انداز محصول

### ۲.۱ بیانیه چشم‌انداز

> برای **کاربران علاقه‌مند به سرمایه‌گذاری در بورس** که به **شفافیت و امنیت** احتیاج دارند،  
> BourseChain یک **سامانه کارگزاری آنلاین** است که **خرید و فروش سهام** را با **ثبت روی بلاکچین** و **اعلان لحظه‌ای** فراهم می‌کند.  
> برخلاف سیستم‌های سنتی، BourseChain **قابلیت ردیابی تراکنش‌ها** را از طریق Smart Contract تضمین می‌کند.

### ۲.۲ ویژگی‌های کلیدی

1. **مشاهده لیست سهام** با قیمت لحظه‌ای و نمودار
2. **ثبت سفارش خرید/فروش** با موتور matching قیمت-زمان
3. **اعلان لحظه‌ای** وضعیت سفارش‌ها (WebSocket)
4. **ثبت تراکنش در بلاکچین** خصوصی EVM
5. **سبد سهام و وضعیت معاملات** با P&L
6. **ورود با اتریوم** (SIWE - Sign-In with Ethereum)

## ۳. محدوده پروژه

### ۳.۱ در محدوده (In Scope)

- فرانت‌اند React با UI دوزبانه (فارسی/انگلیسی)
- API REST با Django REST Framework
- Matching Engine با الگوریتم Price-Time Priority
- Smart Contract برای ثبت تراکنش‌ها
- WebSocket برای اعلان real-time
- Docker و Kubernetes برای deploy
- Monitoring با Prometheus و Grafana

### ۳.۲ خارج از محدوده (Out of Scope)

- Matching Engine روی بلاکچین (غیرمتمرکز)
- اتصال به بورس واقعی ایران
- پرداخت آنلاین (درگاه بانکی)
- موبایل اپلیکیشن native
